Live Measurment Program

Created a program that be able to take a live video feed and measure any object that enters the feed based on a premeasured background object. This project serves as an introduction to machine learning in C++ which I plan to use heavily in later projects. Work in Progress.

Facial Recognition for Smart Mirrors

Program to be used in conjucntion with a smart mirror and JANICE. Allows for a customizable smart mirror to be placed in a central location in a houshold. All members of a household are able to have a custom profile set to display on the mirror that activates as they apporach the mirror based on facial profiling. Work in Progress.
